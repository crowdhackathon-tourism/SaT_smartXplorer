# SaT - smartXplorer

Discover the city beacon by beacon.

## Description

Smart Xplorer offers a smart way to explore and visit a city's sights. Beacon are place all over the city. the user logs in, sets his interests and then starts walking the city's streets and the city speaks. As the user gets near a beacon the application notifies the user of the place he is nearing to. Offering info and rewards if he does visit the site.

## Requirements
- Go!Appzone studio platform.
- iOS developer certificate.
- a beacon
